# 阶段1：创作者内容画像

> 运行频率：仅在初次使用或知识库更新时运行一次
> 输出：《创作者内容画像报告》- 可被后续阶段复用

## 路径约定

- **知识库路径**: 当前工作目录（读取 *.txt 文件作为样本）
- **输出路径**: `./.mirror-writing/profile.md`

## 执行前初始化

确保 `.mirror-writing/` 目录存在，如不存在则创建。

## 样本选择建议

- 数量建议：10-20篇为佳
- 多样性：选择涵盖不同主题/长度的内容
- 代表性：优先选择表现较好的内容
- 完整性：每篇内容应该是完整的

## 分析维度

### 1. 风格特征
- **语气定位**: 励志鸡汤/犀利吐槽/温暖治愈/理性分析/搞笑幽默等
- **人称视角**: 第一/第二/第三人称
- **情感基调与强度**: 积极正向/中性客观/批判反思等
- **亲和力度**: 高冷疏离 vs 亲切近人

### 2. 语言特征
- **高频词汇**: 至少30个词，按类别分组（核心动词、情感词汇、转折词等）
- **标志性口头禅/金句模板**: 如"你要相信..."、"说到底..."
- **句式偏好**: 长短句比例、各类句型使用频率
- **节奏韵律**: 句子节奏、逗号密度、语气词使用

### 3. 内容结构范式（重点！后续会用于跨风格融合）

#### 开头模式库
识别所有常用的开头类型，每种类型提供2-3个典型示例：
- 痛点直击："你是不是也经常..."
- 场景代入："昨天有个朋友问我..."
- 金句开场："真正厉害的人..."
- 数据冲击："90%的人都不知道..."
- 悬念设置："有一件事，我一直没告诉你..."
- 提问开场："你有没有想过..."

#### 正文展开模式
- 并列式/递进式/对比式/故事线/问答式
- 论证方式：举例/类比/数据/权威引用

#### 高潮设置规律
- 核心观点通常出现的位置（前1/3、中间、后1/3）
- 情绪高潮的触发方式

#### 结尾模式库
识别所有常用的结尾类型，每种类型提供2-3个典型示例：
- 金句总结："人生没有白走的路"
- 行动号召："从今天开始，试着..."
- 开放思考："你觉得呢？"
- 情感升华："相信自己，你比想象中更强大"
- 互动引导："评论区告诉我..."

### 4. 主题与内容
- **核心主题**: 3-5个
- **内容标签云**: 20-30个
- **价值主张/世界观**: 创作者传递的核心理念
- **目标受众画像**: 年龄、身份、痛点

### 5. 爆款规律（如能从内容推断）
- 高表现内容的共同特征
- 情绪触发点规律

## 画像更新频率

| 场景 | 建议操作 |
|-----|---------|
| 知识库新增<20%内容 | 无需更新 |
| 知识库新增20-50%内容 | 补充分析 |
| 知识库新增>50%内容 | 重新运行完整分析 |
| 博主风格明显转变 | 重新运行完整分析 |
